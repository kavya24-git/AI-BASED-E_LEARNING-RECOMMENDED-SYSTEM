# AI-BASED-E_LEARNING-RECOMMENDED-SYSTEM
This project is an AI-based machine learning system that recommends
e-learning courses to users based on their interests and behavior.

## Features
- User registration and login
- Course recommendation using ML
- Personalized suggestions

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn

## Project Structure
- app.py – Flask main application
- datasets/ – Dataset files
- preprocess.py – Data preprocessing
- recommendation_model.py – Recommendation logic
- model.pkl – Trained ML model
- templates/ – HTML files
- static/ – CSS and images

## How to Run
pip install -r requirements.txt
python app.py

## Project Type
Academic / Learning Project
